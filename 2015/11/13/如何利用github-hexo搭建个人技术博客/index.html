##3.添加SSH key到Github##

（1）在你的电脑中，打开C:\Users*.ssh目录，其中*为你的电脑账户名，打开id_rsa.pub文件，将里面的内容复制。

（2）登录你的github主页，点击你的头像，点Settings,在网页左边有一个SSH的选项，点进去，点Add SSH key，把你第一小步复制的key添加进去。这样你的SSH key就添加成功了。

##4.在Github中建立repository##

在你的Github主页右上角的头像处点击New repository，命名方式为yourname.github.io，其中yourname为你的Github账户名。然后将“Initialize this repository with a README”选项勾上。点击Create repository即可。
